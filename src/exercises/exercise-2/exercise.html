Exercise 2 - Click!

A button that can't be clicked isn't much of a button is it?

In this exercise, you'll just need to capture the onClick event and re-emit it. This will be important later, as you don't want the component to fire the click event every time it's clicked! Think about why that might be the case.

Don't forget to use some form of preventDefault() to stop the event from bubbling! If you're not familiar with event handling and emission in Vue, you can read up on it here (event handling) and here (event emission).

To check how you did, just run npm run exercise-2-verify.
