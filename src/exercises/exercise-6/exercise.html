Exercise 6 - Special (keyboard) operations

We're at the end of the line! There's only one more thing we HAVE to do to make this button fully accessible.

If you recall, in Exercise 3 we mentioned that screen reader users tend to navigate around the page with the keyboard, and they can focus interactive elements on the page with the Tab key. If screen reader users are just using the keyboard then, how can they click the button?

Well, it turns out an accessible button can be activated via both the Space bar and Enter key. While the button is focused, it should fire the "click" event if either enter or the space bar is pressed and it's not disabled. This Vue documentation on event listener modifiers should also come in handy.

Go ahead and implement support for Space and Enter on your button, then verify your progress with npm run exercise-6-verify. We will be verifying your support via the keyup event, so keep that in mind.
